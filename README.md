# Monocular Depth Estimation Depth Anything V2

This example demonstrates monocular depth estimation using the Depth Anything V2 model. It programmatically generates a synthetic 3D scene (with foreground and background objects), processes it using the Hugging Face Transformers pipeline, and saves a side-by-side comparison of the original image and the generated depth map.

## Language

`python`

## How to Run

pip install torch transformers pillow numpy
python main.py
